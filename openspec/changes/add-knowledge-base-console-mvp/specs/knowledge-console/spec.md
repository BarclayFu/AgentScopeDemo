## ADDED Requirements

### Requirement: 知识库管理台
系统 MUST 提供一个可用的知识库管理台页面，用于查看和维护知识库条目。

#### Scenario: 成功加载知识库列表
- **WHEN** 管理员打开知识库管理页
- **THEN** 前端调用知识库列表接口并展示知识条目、总数和最近更新时间

#### Scenario: 空知识库状态
- **WHEN** 当前没有任何可展示的知识条目
- **THEN** 页面显示空数据提示并保留新增入口

#### Scenario: 加载失败
- **WHEN** 知识库管理接口请求失败
- **THEN** 页面显示错误提示并提供重试入口

### Requirement: 知识条目管理接口
系统 MUST 提供结构化的知识条目管理接口，用于新增、删除和查询知识条目。

#### Scenario: 新增知识条目
- **WHEN** 管理员提交标题和内容
- **THEN** 服务创建新的知识条目并返回明确结果

#### Scenario: 删除知识条目
- **WHEN** 管理员删除指定知识条目
- **THEN** 服务删除该条目并返回成功结果

#### Scenario: 查询知识条目
- **WHEN** 前端调用知识条目列表接口
- **THEN** 服务返回条目列表和总数等元数据

### Requirement: 知识库状态与刷新
系统 MUST 提供知识库状态查询和刷新能力。

#### Scenario: 查看知识库状态
- **WHEN** 前端调用状态接口
- **THEN** 服务返回知识库状态、条目数、最近更新时间和最近一次操作结果

#### Scenario: 手动刷新知识库
- **WHEN** 管理员触发知识库刷新或重建
- **THEN** 服务执行刷新动作并返回明确状态
